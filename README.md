<p><strong>To run on your local machine:</strong></p>

```
git clone https://github.com/VERSAYANA/pomodoro-clock.git
cd pomodoro-clock
yarn install
yarn start
```

or

```
git clone https://github.com/VERSAYANA/pomodoro-clock.git
cd pomodoro-clock
npm install
npm start
```
